# 크로마 스텝 (Chroma Step)

Three.js 기반 웹 리듬 게임

## 실행 방법
- index.html 파일을 브라우저에서 직접 열면 바로 플레이 가능

## 배포(GitHub Pages)
- main 브랜치의 index.html을 사용하여 GitHub Pages로 배포
- 배포 주소: https://namseokyoo.github.io/chroma_step/

## 주요 특징
- 3D 타일 기반 리듬 게임
- 콤보, 미션, 테마 시스템
- 완전 클라이언트 사이드 동작
